Used Car Price Prediction

Developed machine learning regression models to estimate used vehicle prices based on car characteristics, helping buyers, sellers, and dealerships make more informed pricing decisions.

Python Machine Learning Regression Scikit-Learn Feature Engineering

+ Used Car Price Prediction +

Overview

The used car market is highly dynamic, with vehicle prices influenced by numerous factors including age, mileage, brand, fuel type, and engine specifications. Accurately estimating a vehicle's market value can help dealerships improve pricing strategies while helping buyers avoid overpaying.

This project applies machine learning regression techniques to predict used car prices based on vehicle characteristics. The objective was to identify the factors that most strongly influence resale value and develop a predictive model capable of generating reliable price estimates.

+ Project Process +

Step 1: Data Collection and Understanding

The dataset was built using web scraping techniques to collect vehicle listings from Cars.com. After collection, the final dataset contained 4,009 used vehicle records.

The dataset included information such as:

  • Brand & Model
  • Model Year
  • Mileage
  • Fuel Type
  • Engine Type
  • Horsepower
  • Transmission
  • Exterior Color
  • Interior Color
  • Accident History
  • Clean Title Status
  • Price (Target Variable)

Before beginning any modeling work, I explored the variables to understand how different vehicle characteristics might influence resale value.

Step 2: Data Cleaning and Exploratory Data Analysis

Like most real-world datasets, the raw data required preprocessing before it could be used for machine learning. I performed the following steps:

  • Handled missing values and incomplete vehicle records.
  • Checked for duplicate listings.
  • Validated numerical variables such as mileage, horsepower, and price.
  • Explored distributions of key variables using visualizations.
  • Investigated relationships between mileage, model year, and vehicle prices.
  • Identified potential outliers and unusual pricing patterns.

Exploratory analysis revealed that newer vehicles generally commanded higher prices, while increased mileage was associated with lower resale values.

Step 3: Feature Engineering

To improve model performance, several preprocessing and feature engineering steps were performed.

  • Categorical Encoding: Vehicle characteristics such as brand, fuel type, transmission, and colors were converted into machine-readable formats.
  • Average Model Price Feature: Created an additional feature representing average pricing by model to capture brand and model-specific value trends.
  • Feature Scaling: Applied where appropriate to prepare data for model training.

These engineered features helped the algorithms better capture pricing patterns within the used car market.

Step 4: Model Selection and Training

Several regression algorithms were tested and compared to determine which model could most accurately predict vehicle prices.

  1. Linear Regression
    Used as a baseline model to establish initial predictive performance.
  2. Random Forest Regressor
    An ensemble learning method that combines multiple decision trees to improve predictive accuracy and reduce overfitting.
  3. Gradient Boosting Regressor
    A powerful boosting algorithm that sequentially improves predictions by correcting previous model errors.

Each model was trained using the same dataset and evaluated using standard regression performance metrics.

Step 5: Model Evaluation

After training, model performance was compared to determine which algorithm best captured the pricing dynamics of the used car market.

Among all tested models, the Random Forest Regressor emerged as the strongest performer.

Its ability to model complex, non-linear relationships allowed it to outperform traditional linear methods and better capture the interactions between vehicle characteristics and price.

Step 6: Understanding What Drives Vehicle Prices

One of the advantages of tree-based models is their ability to provide feature importance scores.

The Random Forest model identified the following variables as the most influential factors affecting vehicle prices:

  • Mileage: 34.6%
  • Model Year: 10.8%
  • Exterior Color Encoding: 8.2%
  • Horsepower: 6.7%
  • Other Features: Contributed the remaining influence.

Mileage was by far the most important predictor, highlighting the strong impact of vehicle usage on resale value.

Business Relevance

This project demonstrates how machine learning can be applied to solve pricing challenges in the automotive market.

  • Dealerships can use predictive pricing models to improve inventory valuation.
  • Sellers can price vehicles more competitively.
  • Buyers can compare listing prices against model estimates to identify overpriced vehicles.
  • Fleet Managers can estimate depreciation and future resale values.

What I Learned

  • Building end-to-end machine learning workflows.
  • Collecting and preparing real-world data through web scraping.

British Airways Customer Booking Prediction

This project analyzes customer booking behavior for British Airways to identify the factors that influence booking completion and to develop predictive models capable of estimating the likelihood that a customer will finalize a booking.

Python Pandas Matplotlib Seaborn EDA

+ British Airways Customer Booking Prediction +

Overview

Airlines invest heavily in marketing campaigns to attract potential customers to their booking platforms. However, not every visitor who begins the booking process ultimately completes a reservation.

The analysis combines exploratory data analysis, statistical testing, customer segmentation, and machine learning techniques to uncover actionable insights that can improve conversion rates and support more efficient marketing strategies.

Data Preparation

Several preprocessing steps were performed before analysis:

  • Missing value assessment
  • Duplicate record validation.
  • Categorical variable encoding
  • Evaluated multicollinearity using Variance Inflation Factors (VIF).
  • Data normalization where appropriate
